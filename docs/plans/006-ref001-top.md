# 006 REF001 トップ

## 目的

ログイン後の起点画面として、ロールに応じたメニュータイルを表示する(docs/screens.md REF001)。

## 前提(依存するplan)

- 003 共通レイアウト・UI部品(`Tile`)
- 005 AUTH001 ログイン画面

## 実装内容

- `app/(authenticated)/page.tsx`: セッションのロールを見てタイルを出し分けるServer Component
  - 私の経歴書タイル: 一般社員/管理職のみ(人事・営業には非表示)
  - 経歴書一覧・スキルマップタイル: 全ロール
  - アカウント一覧・マスタ管理タイル: 管理職のみ
- 一般メニューと管理者メニューを視覚的に分離し、管理者メニューは「管理者メニュー(管理職のみ)」見出し付きの枠+淡い背景のグループにまとめる(`components/home/`)
- 遷移先が未実装の間はタイルを「準備中」(hrefなし)で表示する

## 受け入れ基準

- 3ロールそれぞれでログインした際、docs/screens.md REF001 の表示条件どおりにタイルが出る
- 管理者メニューが見出し付きグループとして分離表示される

## 検証方法

1. Playwrightで3ロールそれぞれのタイル表示を確認する
2. `npm run verify`が通ることを確認する
